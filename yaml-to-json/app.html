使用提示与常见错误

YAML 语法规则

YAML 使用缩进表示层级关系，冒号表示键值对。

name: BenniaoOffice
version: 1.0
services:
  web:
    port: 8080

常见错误

缩进错误：YAML对缩进敏感，必须使用空格，不能使用Tab。

缺少冒号：键值对必须用冒号分隔。

引号不匹配：字符串引号必须成对出现。

转换说明

YAML注释（#）在转换到JSON时会被忽略。

YAML的锚点（&）和别名（*）将被解析为相应的JSON结构。

多行字符串（| 或 >）将转换为JSON字符串。
