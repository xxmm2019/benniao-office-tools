TACCESS 转 NGINX 规则转换工具

注意：本工具不能保证 100% 准确转换所有规则，对于生产环境请务必进行充分测试

关于这个工具

在线将 Apache .htaccess 伪静态规则转换为 Nginx 重写规则，支持重定向、RewriteRule、条件判断。

使用场景

    1. 转换后的 Nginx 规则通常需要放置在 server {} 块内，或使用 include 指令引入

    2. Nginx 不支持 .htaccess 文件，所有规则必须写入主配置文件或虚拟主机配置文件中

    3. 转换后请仔细检查规则，特别是正则表达式，可能需要根据 Nginx 语法进行调整

    4. 某些 Apache 特有指令（如 php_value）在 Nginx 中需要在不同的位置配置
