工具介绍

这是一个轻量级的 JSON 到 YAML 转换工具。所有转换都在您的浏览器中完成，数据不会发送到任何服务器，确保您的隐私安全。

主要功能

  • 实时将 JSON 格式转换为 YAML 格式
  • 支持复杂嵌套结构的转换
  • 一键复制转换结果到剪贴板
  • 显示转换错误信息
  • 完全离线工作，无需网络连接

使用方法

  1. 点击上方"打开工具"按钮
  2. 在左侧输入框中粘贴或输入 JSON 数据
  3. 右侧将自动显示转换后的 YAML 格式
  4. 点击"复制"按钮将结果复制到剪贴板
