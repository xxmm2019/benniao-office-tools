YAML 语法提示

YAML 使用缩进表示结构，注意使用空格（推荐2个或4个空格），不要使用制表符(Tab)。

# 正确示例 person: name: "张三" age: 30 address: city: "北京" postal: "100000"

常见错误

1. 缩进不一致：必须使用相同数量的空格缩进同一级别的元素

2. 冒号后缺少空格：key: value 正确 | key:value 错误

3. 字符串引号使用不当：特殊字符需要引号包裹

XML 输出说明

转换后的 XML 将使用 UTF-8 编码，并自动格式化缩进。

数组元素会自动添加索引属性：item index="0"

<?xml version="1.0" encoding="UTF-8"?> <root> <person> <name>张三</name> </person> </root>
